# Firebase

## Authentication

- [Firebase Console ｜ Google](https://console.firebase.google.com/?hl=ja) にアクセス
- Google サインインする
- [Authentication] - [始める]をクリック
- [Sign-in method] - [メール／パスワード]をクリック
- [有効にする]を ON にして[保存]をクリック
